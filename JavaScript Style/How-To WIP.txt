How-To Documentation
Work in progress

1. Install Node.js: https://nodejs.org/en/download/
2. Install the Kinect SDK: https://www.microsoft.com/en-us/download/details.aspx?id=44561
3. Clone GitHub repository: git clone https://github.com/donnerlt/CSCI462.git <location>
4. Install necessary items in node.js: http://www.webondevices.com/get-started-with-xbox-kinect-2-javascript-development/
5. In node.js  command prompt, change directory to wherever you cloned the github repository then change directory to JavaScript Style\Kinect\kinect sample 
6. In noded.js command prompt run the program: node Kinect.js
